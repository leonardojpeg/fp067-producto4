# App NBA - Entrega Final

## Instrucciones para ejecutar:
1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar las dependencias.
3. Ejecutar `npx expo start` para lanzar el proyecto.
4. Pulsar `w` para abrir en el navegador.

## Características:
- Conexión con Firebase Realtime Database.
- Navegación entre lista de jugadores, detalle y reproductor.
- Reproductor de YouTube integrado con controles de interacción.
